## 登录

* 功能:验证用户身份进入系统

* url:"/login"

* 请求方式:POST

* 请求参数

  | 参数     | 类型 | 说明                     |
  | -------- | ---- | ------------------------ |
  | usr      | int  | 用户工号                 |
  | psw      | int  | 用户密码                 |
  | optradio | int  | 用户类型 1:教师 2:管理员 |

  示例:

  {usr: 123456,psw: 123456,optradio: 1}

* 返回字段

  | 返回字段 | 字段类型 | 说明                             |
  | -------- | -------- | -------------------------------- |
  | code     | int      | 返回请求处理状态 200:成功 0:失败 |
  | message  | String   | 返回错误信息                     |
  | usrname  | String   | 用户名                           |

  示例:

  {

   "code":200,

   "message":"ok"

   "usrname":"张三"

  }
